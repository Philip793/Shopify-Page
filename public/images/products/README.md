# Product Images

This folder contains product images for the Magestic board games store.

## Image Naming Convention
- Use descriptive names: `{product-id}-{product-name}.{extension}`
- Example: `1-elegant-board-game-set.jpg`
- Example: `2-strategy-master.jpg`

## Supported Formats
- JPG, PNG, WebP
- Recommended: WebP for better compression

## Image Sizes
- Thumbnails: 400x300px
- Main images: 800x600px
- Gallery: 800x600px
